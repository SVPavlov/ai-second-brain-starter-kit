## How this vault is laid out

Numbered files at the root are the working memory the AI loads first. Capitalized folders are long-term memory.

| File | What it is | Maintained by |
|------|-----------|---------------|
| `00-START-HERE.md` | This map + the catalog | AI |
| `01-me.md` | Who you are, your clients, your recurring work | AI + you |
| `02-command-center.md` | Your daily cockpit: what changed, what's due, what needs you | AI (refresh on "daily brief") |
| `03-tasks.md` | Open loops, your action list | AI |
| `04-log.md` | Append-only trail of what the brain did | AI |
| `05-synthesis.md` | The daily questions the brain needs you to answer, plus friction + run log | AI |

| Folder | Holds |
|--------|-------|
| `Inbox/` | Raw drops, unprocessed (you put things here) |
| `Meetings/` | One note per meeting |
| `People/` | One note per person |
| `Clients/` | One folder per client, each with a `_index.md` hub |
| `Projects/` | Non-client work |
| `Patterns/` | Reusable things that worked, for the next engagement |
| `Sources/` | Cold storage: ingested documents kept for reference |
| `Archive/` | Stale or superseded material (out of the catalog) |
